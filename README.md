# Upskill Web Development Task 4: Build a Responsive Personal Portfolio Website
Task Description:
Create a responsive website  using HTML and CSS (Flexbox & Media Queries) that looks good on mobile, tablet, and desktop screens.

Requirements:
1. Structure (HTML)
Create the following sections:
Header (with your logo and navigation links)

About Me (short paragraph + image)

Cards section

Contact (email or contact form)

Footer

2. Style (CSS) - Look polished and professional
Use Flexbox to arrange layout components (especially for the card section).

Apply Media Queries to adjust the layout using desktop-first or mobile-first approac.

Ensure font sizes, spacing, and images adjust properly.

Add hover effects for interactivity.
